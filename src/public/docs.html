API de Gestão de Alunos

Descrição da API

Esta API permite gerir informações sobre alunos, incluindo criação, leitura, atualização e eliminação de dados. É ideal para aplicações de gestão escolar e outras onde seja necessário manter um registo de alunos.

Iniciar o Servidor

  1. Certifique-se de que o Node.js está instalado no sistema.
  2. Clone o repositório do projeto.
  3. Instale as dependências com o comando: npm install
  4. Inicie o servidor com: npm start
  5. A API ficará disponível em http://localhost:3000

Estrutura da Base de Dados

A aplicação utiliza um ficheiro JSON como base de dados, onde os dados dos alunos são armazenados. O ficheiro é atualizado automaticamente com cada operação de criação, atualização ou eliminação.

Endpoints da API

  • GET /alunos - Retorna todos os alunos registados
  • GET /alunos/:id - Retorna um aluno específico, identificado pelo seu ID
  • POST /alunos - Cria um novo aluno
  • PUT /alunos/:id - Atualiza as informações de um aluno
  • DELETE /alunos/:id - Remove um aluno da base de dados

Exemplos de Uso

Obter todos os alunos

curl http://localhost:3000/alunos

Obter um aluno específico

curl http://localhost:3000/alunos/1

Criar um aluno


curl -X POST -H "Content-Type: application/json" -d '{
  "nome": "João",
  "curso": "Informática",
  "ano": 2001
}' http://localhost:3000/alunos

Atualizar um aluno


curl -X PUT -H "Content-Type: application/json" -d '{
  "nome": "Maria",
  "curso": "Engenharia",
  "ano": 2002
}' http://localhost:3000/alunos/1

Remover um aluno

curl -X DELETE http://localhost:3000/alunos/1
